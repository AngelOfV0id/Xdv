Xdv - extreme directory navigator

Source code / Исходный код
Releases / Выпуски

NOT a File Manager, but a DIRECTORY NAVIGATOR. A tiny and blazingly fast utility for quick navigating in Windows file system, viewing directories and opening files.

Simple and minimalistic command-line interface. Easy to remember keyboard commands.
Lightweight, portable - just download and run.
Written in C. Powered by Win32 API. Less than 120 lines of code.

Readme  /  Changelog


НЕ файловый менеджер, а НАВИГАТОР ДИРЕКТОРИЙ. Минималистичная, легковесная, шустрая утилита для навигации по файловой системе ОС Windows со скоростью света (ну почти).

Простой и минималистичный интерфейс командной строки (CLI). Простое и удобное управление с клавиатуры.
Легковесность, портативность - установка не требуется.
Менее 120 строк кода на C (с щепоткой магии Win32 API).
